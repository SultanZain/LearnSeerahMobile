# Contributing Guide

Thank you for contributing!

## Workflow

1. Fork the repo or create a feature branch from `dev`.
2. Make your changes.
3. Open a pull request **into `dev` branch**.
4. Once tested and reviewed, changes will be manually merged into `main`.
